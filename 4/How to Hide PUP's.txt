How to Hide "PUP" -Potentially Unwanted Programs-

USING "DB Browser for SQLite.exe"

1- Play "DB Browser for SQLite.exe"
2- Choose the "app.db" -Database- file
3- Click Browse Data
4- Select the Table "tbl_appbrowse_0413713369"
5- Scroll to the right side until you see "visible" Cell
6- If the Filter is "0" it means hidden & If it is "1" it means Visible
7- Save it and replace it from PS4 Internal Storage \/ \/

You Can Find The "app.db" File In "system_data/priv/mms" PS4 Internal
And to Get It You Need To Use FTP
